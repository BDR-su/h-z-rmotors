# HIZIR MOTORS Website

Modern, responsive website for HIZIR MOTORS - A motorcycle repair and maintenance service provider. This website serves as a digital catalog to showcase services and products.

## Features

- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Dark theme with neon blue accents
- **Service Showcase**: Display your services with icons and descriptions
- **Product Catalog**: Showcase your products with images and details
- **Contact Form**: Easy way for customers to get in touch
- **Interactive Elements**: Smooth scrolling, animations, and hover effects
- **SEO Optimized**: Built with search engine optimization in mind

## Pages

1. **Homepage** - Introduction, services, featured products, about us, and contact information
2. **Products** - Detailed product catalog (to be expanded)
3. **Services** - Detailed service information (to be expanded)
4. **About Us** - Company information and team (to be expanded)
5. **Contact** - Contact form and location map

## Setup Instructions

1. Clone or download this repository
2. Open `index.html` in a web browser to view the website locally
3. For development, you can use a local server like Live Server in VS Code

## Customization

1. **Content**: Edit the HTML files to update text, images, and links
2. **Styling**: Modify the CSS in `css/style.css` to change colors, fonts, and layout
3. **Functionality**: Update JavaScript in `js/main.js` for interactive features
4. **Images**: Replace placeholder images in the `images/` folder with your own

## Image Requirements

- **Logo**: Recommended size 200x100px (PNG with transparent background)
- **Hero Image**: Recommended size 1920x1080px (high-quality JPG/PNG)
- **Product Images**: Recommended size 800x800px (square aspect ratio, JPG/PNG)
